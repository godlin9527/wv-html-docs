B 站知识型 KOL：复刻性价比筛选

这页不是“百大名单大全”，而是从 B 站百大/百大级知识区创作者里，筛哪些表达方式值得我们学习和复刻。核心问题：普通人做观点表达、工作汇报、知识讲解时，能不能用 HyperFrames 低成本做出类似效果？

先说结论 候选筛选 优先拉片 代表样本 下一步

先说结论

B 站知识区百大里，真正适合我们“低素材 faceless / PPT-like / HyperFrames 复刻”的，不一定是最有名的那些。很多百大强在真人人格、外拍、历史素材、采访、梗图和长期选题积累。我们要优先找的是：图解能力强、画面系统简单、素材依赖可控、信息结构清楚的创作者。
最值得学

小约翰可汗、思维实验室、一数、半佛仙人的部分视频。它们都有强叙事或强结构，能拆出图解/卡片/时间线方法。

只能学脚本

小Lin说、罗翔说刑法、宋浩老师、部分真人科普。内容组织好，但画面强依赖真人或讲台。

暂不适合复刻

芳斯塔芙、Chubbyemu、何同学等依赖大量真实素材、实验、外拍或高制作包装的案例。

建议优先拉片顺序

小约翰可汗：看强叙事如何用低复杂度图形撑住

不是照搬他的梗，而是学：人物如何登场、地点如何切换、地图和标签如何服务叙事、每隔多久给一次视觉信息点。

思维实验室：看严肃长议题如何章节化

适合研究“长逻辑链怎么被拆成段落”。如果我们以后做 AI、商业、社会议题，这类结构比纯搞笑 KOL 更可迁移。

半佛仙人：看观点型内容如何有攻击性

适合学标题、金句、反共识、段落转折。但视觉不能照搬梗图堆叠，否则会变乱。

一数：看抽象概念如何最小图形化

如果我们要做“概念解释”，一数式推导很值得学：每一帧只新增一个变量或关系。

可先看的代表样本

这里先嵌入能检索到的公开 YouTube 镜像样本，后续如果要精拉 B 站原片，可以再用 B 站源下载和抽帧。

小约翰可汗：硬核狠人类样本
强旁白 + 历史图像 + 地图/标签 + 节奏梗

适合学叙事密度和视觉锚点，但需要警惕素材、事实和语气依赖。

思维实验室：国际关系长叙事样本
严肃议题 + 地图 + 资料图 + 章节化

适合研究“长逻辑如何不散”，比很多娱乐化知识视频更适合做结构参考。

半佛仙人：商业观点短样本
口语观点 + 截图/梗图 + 快速段落

适合学“观点如何更有传播性”，但视觉上需要重新收敛成更干净的卡片系统。

小Lin说：财经商业讲解样本
真人主持 + 图表 + b-roll + 生活化例子

适合学脚本和例子，不适合作为低成本 faceless 视觉模板。
