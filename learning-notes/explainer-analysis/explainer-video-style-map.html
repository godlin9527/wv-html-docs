风格标准

优先使用行业文章、制作公司、课程或案例库中真实出现的英文说法，不使用我们临时发明的分类名。

分类层级

有些是视觉语言，有些是生产方式、媒体体裁或动画技术；同一个案例可以跨类，但 core 只代表它最能定义的一面。

核心代表作

优先选择被多个案例库引用、由标杆创作者/品牌发布、或能清楚代表该风格方法的可播放案例。

补充参考

用于补齐细分方向、中文语境或生产方式。它们有参考价值，但不等同于该风格最经典样本。

HyperFrames 复刻性价比

这个排序不是行业地位排序，而是制作选型排序：如果目标是用 HyperFrames 复刻经典风格，同时兼顾难度、效果和可量产性，建议按下面顺序练习。

判断维度：复刻难度 / 表现效果 / 可复用组件 / 对 HyperFrames 的适配度 原则：先做可控、可复用、能快速产出专业感的类型，再挑战高资产、高制作难度的风格。
优先级 风格 难度 效果 性价比 为什么适合 / 不适合先做
01 Screencast / Software Demo / UI Animation 低-中 中-高 最高 最适合 HyperFrames：屏幕录制、浏览器框、缩放、高亮、箭头、步骤卡、章节标题都能组件化。适合先建立可量产的教程型 explainer 基础。
02 Animated Presentation / Presentation-style 低-中 中 很高 接近动态课件，但比 PPT 更强调 reveal、停顿、镜头移动和信息节奏。HyperFrames 可以快速复用标题页、概念卡、流程图、对比图和列表组件。
03 Typography / Kinetic Typography 中 高 高 适合做开头爆点、金句、转场和情绪段落。全片都做文字动效会累，但作为模块非常有传播感，也容易用 CSS/SVG/GSAP 控制。
04 Infographic Video / Data Visualization 中 高 高 适合解释观点、数据、比例和趋势。难点在信息设计，不是资产绘制；一旦建立图表、数字、比例尺组件，复用价值很高。
05 Motion Graphics / Graphic Animation 中-高 高 中高 最像传统 MG 的核心能力，适合流程、结构、抽象概念。难点是每个镜头都要有清楚的运动语义，不能只是图形乱飞。
06 Plain English / Common Craft Cutout 中 中高 中高 风格强、解释清楚，但真正的手工纸片感需要纹理、阴影、手部动作和低保真构图。可以用 HTML/SVG 仿，但要避免变成普通扁平插画。
07 Whiteboard Animation / Visual Scribing 中-高 中高 中 经典、清楚，但需要大量线稿路径、绘制顺序和画布组织。HyperFrames 能做，但产能不如软件演示和动态课件。
08 Editorial Explainer / Explainer Journalism 高 高 中 难点主要在资料、观点、证据链和叙事结构，不只是动画。适合成熟后做，但不适合作为第一批复刻训练。
09 Live Action / Live Action + Animation 高 高 中低 效果依赖真人拍摄、表演、场景和剪辑。HyperFrames 可以加动画包装，但不能替代拍摄资产本身。
10 2D Animation / Illustrated Animation 高 高 中低 需要原创插画、角色资产、镜头调度和一致世界观。适合做高完成度作品，不适合快速复刻大量案例。
11 Stop Motion 很高 中高 低 真正的定格动画依赖真实材料和逐帧拍摄；如果只用 HTML 模拟，很容易失去手工质感。更适合作为风格参考，不适合早期复刻。
12 3D Animation / Scientific & Product CG 很高 很高 低 视觉上限高，但需要 3D 建模、材质、灯光、相机和渲染。HyperFrames 可承载 Three.js，但复刻经典 CG 的成本远高于当前收益。
